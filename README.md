# Travassyx Docs

Documentação pública da Travassyx Platform para comunidade, clientes e equipes técnicas que precisam instalar, ativar, conectar e operar a plataforma.

## O Que É a Travassyx Platform

A Travassyx Platform é uma plataforma de observabilidade operacional, governança e FinOps para ambientes híbridos e multi-cloud.

Ela ajuda equipes de infraestrutura, NOC, gestão e FinOps a responder:

- quais recursos existem no ambiente;
- qual é a saúde operacional;
- quais riscos exigem atenção;
- onde há desperdício;
- qual custo real ou estimado está associado ao ambiente;
- quais incidentes, evidências e runbooks devem orientar a tratativa.
